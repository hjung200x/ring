# ring

V1 service for collecting KEIT announcements, extracting notice documents, scoring them against saved interest profiles, and showing matched notifications in a web app.

## Workspaces
- `apps/api`: Fastify + Prisma backend
- `apps/web`: React + Vite frontend
- `packages/schemas`: shared zod schemas and DTOs
- `packages/types`: shared domain types
